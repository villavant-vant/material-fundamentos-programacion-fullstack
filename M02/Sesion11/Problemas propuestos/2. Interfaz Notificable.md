# Problema propuesto 2: Notificaciones de la biblioteca

Crea una interfaz `Notificable` con el método `enviarNotificacion(String mensaje)`.

Crea clases `Email`, `SMS` y `Push` que implementen la interfaz, cada una imprimiendo el mensaje con un formato distinto (por ejemplo, `"[EMAIL] " + mensaje`).

En el sistema de biblioteca, cuando un préstamo esté vencido (ver problema propuesto 1), notifica al usuario usando polimorfismo: recorre una lista de `Notificable` (que puede mezclar `Email`, `SMS` y `Push`) y llama a `enviarNotificacion()` en cada una con el mismo mensaje de aviso.
