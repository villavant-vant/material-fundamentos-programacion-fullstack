# Problema propuesto 1: Clase CuentaBancaria

Crea una clase `CCuentaBancaria` con atributos **privados**: `titular`, `numeroCuenta` y `saldo`.

Incluye:
- Un constructor que reciba `titular` y `saldo` inicial (el número de cuenta se genera automáticamente).
- Un método `depositar(double monto)` que aumente el saldo (validando que el monto sea positivo).
- Un método `retirar(double monto)` que disminuya el saldo, validando que no se sobregire la cuenta.
- Un método `mostrarSaldo()` que imprima el titular y el saldo actual.

En una clase `Principal`, crea 2 cuentas con distinto saldo inicial y prueba depósitos y retiros, incluyendo un intento de retiro que exceda el saldo disponible.
