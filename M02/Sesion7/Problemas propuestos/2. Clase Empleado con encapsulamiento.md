# Problema propuesto 2: Clase Empleado con encapsulamiento

Crea una clase `CEmpleado` con atributos privados: `nombre`, `sueldoBase` y `añosTrabajados`.

Incluye:
- Getters y setters para cada atributo (el setter de `sueldoBase` no debe permitir valores negativos).
- Dos constructores: uno que reciba solo el nombre (con sueldo base y años en 0 por defecto), y otro que reciba los 3 datos.
- Un método `calcularSueldoFinal()` que agregue un 2% del sueldo base por cada año trabajado.

En una clase `Principal`, crea 3 empleados usando ambos constructores y muestra el sueldo final calculado de cada uno.
