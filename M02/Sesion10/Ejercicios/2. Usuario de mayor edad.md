# Ejercicio 2: Usuario de mayor edad

1. Ingresar la cantidad de usuarios.
2. En cada iteración, solicitar el nombre, edad y género de cada usuario y guardarlo como un objeto `CUsuario` en un `ArrayList<CUsuario>`.
3. Imprimir la lista de los nombres de los usuarios almacenados.
4. Imprimir el nombre del usuario que tiene mayor edad.

Usa la clase [Clases/3. CUsuario.java](../Clases/3.%20CUsuario.java) como modelo.
