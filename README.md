# Material - Fundamentos de Programación Full Stack

Material de clase, ejercicios y problemas propuestos del curso **Fundamentos de Programación Full Stack** de UTEC, desarrollado en Java.

## Estructura del repositorio

El contenido está organizado por módulo y sesión. Cada sesión contiene tres carpetas:

- **Clases**: ejemplos de código usados durante la clase.
- **Ejercicios**: ejercicios resueltos en sesión.
- **Problemas propuestos**: problemas para practicar de forma autónoma.

### M01 - Fundamentos de Java

| Sesión | Tema |
| --- | --- |
| Sesion1 | Introducción a Java: entrada y salida de datos, `Scanner` |
| Sesion2 | Variables, tipos de datos, operadores aritméticos y lógicos |
| Sesion3 | Estructuras condicionales: `if`, anidados, ternario, `switch` |
| Sesion4 | Estructuras repetitivas: `while`, `do-while`, `break`, `continue` |
| Sesion5 | Métodos y alcance de variables (scope) |
| Sesion6 | Arreglos (arrays) |

### M02 - Programación Orientada a Objetos

| Sesión | Tema |
| --- | --- |
| Sesion7 | Introducción a clases y objetos |
| Sesion8 | Relaciones entre clases: asociación, agregación, composición; `ArrayList` |
| Sesion9 | Herencia |
| Sesion10 | Listas y operaciones CRUD |
| Sesion11 | Proyecto integrador (biblioteca) |

## Requisitos

- JDK 8 o superior instalado.
- Un IDE o editor con soporte para Java (VS Code, IntelliJ IDEA, Eclipse, etc.).

## Soluciones por sesión

Además de `main`, el repositorio cuenta con una rama de solución por cada sesión, con el nombre `<Módulo>-<Sesión>-solution`:

- `M01-Sesion1-solution`
- `M01-Sesion2-solution`
- `M01-Sesion3-solution`
- `M01-Sesion4-solution`
- `M01-Sesion5-solution`
- `M01-Sesion6-solution`
- `M02-Sesion7-solution`
- `M02-Sesion8-solution`
- `M02-Sesion9-solution`
- `M02-Sesion10-solution`

Cada rama contiene la resolución de los ejercicios y problemas propuestos de esa sesión. Para revisarla:

```bash
git checkout <Módulo>-<Sesión>-solution
```
