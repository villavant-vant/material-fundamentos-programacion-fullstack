# Problema propuesto 1: Clasificador de IMC

Solicita el peso (kg) y la estatura (m) de una persona. Calcula el índice de masa corporal:

```
IMC = peso / (estatura * estatura)
```

Usando `if/else if`, clasifica el resultado:

- IMC < 18.5: "Bajo peso"
- 18.5 <= IMC < 25: "Normal"
- 25 <= IMC < 30: "Sobrepeso"
- IMC >= 30: "Obesidad"
