# Ejercicio 6: Calculadora con switch

Solicita 2 números enteros y un operador (`+`, `-`, `*`, `/`). Usando `switch`, imprime el resultado de la operación.

Si el usuario selecciona la división, valida que el segundo número no sea 0; si lo es, muestra el mensaje `"NO SE PUEDE DIVIDIR ENTRE 0"` en vez de realizar la operación.
