# Ejercicio 5: Identificar billete

Usando `switch`, solicita el valor de un billete (entero) y:

- Si ingresa un billete de 10, 20 o 50, imprime `"Billetes basicos"`.
- Si ingresa un billete de 100, imprime `"Tiene dinero"`.
- Si ingresa un billete de 200, imprime `"Has recibido tu grati"`.
- Caso contrario, imprime `"El billete no existe"`.

Pista: usa agrupación de `case` (fall-through) para los billetes básicos.
