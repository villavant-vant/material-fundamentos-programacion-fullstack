# Problema propuesto 2: Encuesta con conteo de respuestas

Se realizó una encuesta donde cada persona respondió con un número del 1 al 3 (1=Sí, 2=No, 3=Tal vez). Los resultados están en el arreglo:

```java
int[] respuestas = {1, 2, 1, 3, 1, 2, 2, 1, 3, 1};
```

Recorre el arreglo y cuenta cuántas respuestas hay de cada tipo. Imprime el resultado como:

```
Si: 4
No: 3
Tal vez: 2
```
