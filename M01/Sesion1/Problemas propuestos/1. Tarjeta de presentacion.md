# Problema propuesto 1: Tarjeta de presentación

Declara variables para tu nombre completo, tu edad, tu ciudad y tu lenguaje de programación favorito. Imprime una "tarjeta de presentación" usando `System.out.println` y `printf`, con un formato similar a:

```
==================
   Jorge Villavicencio
==================
Edad: 32 años
Ciudad: Lima
Lenguaje favorito: Java
```

No se pide capturar datos por teclado, solo declarar las variables y mostrarlas con buen formato.
