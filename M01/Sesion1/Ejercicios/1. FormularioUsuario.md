# Ejercicio 1: Formulario de usuario

Crea un programa que solicite por teclado los siguientes datos:

- Nombre (texto)
- Edad (entero)
- Nota final (decimal)
- ¿Le gusta programar? (true/false)

Luego, muestra todos los datos usando `printf`.
