# Problema propuesto 1: Menú interactivo

Crea un menú interactivo en consola usando un ciclo `while` que se repita hasta que el usuario elija salir:

```
1. Sumar dos numeros
2. Ver tabla de multiplicar de un numero
3. Calcular factorial de un numero
4. Salir
```

Según la opción elegida, solicita los datos necesarios, muestra el resultado y vuelve a mostrar el menú, hasta que el usuario seleccione la opción 4 (Salir).
