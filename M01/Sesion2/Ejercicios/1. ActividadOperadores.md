# Ejercicio 1: Perfil de una persona (variables)

Declara variables de distintos tipos para representar el perfil de una persona:

- Nombre (`String`)
- Edad (`byte`)
- Altura (`float`)
- ¿Es estudiante? (`boolean`)
- Género (`char`)

Imprime cada dato con una etiqueta descriptiva (por ejemplo: `"Nombre: " + nombre`).
