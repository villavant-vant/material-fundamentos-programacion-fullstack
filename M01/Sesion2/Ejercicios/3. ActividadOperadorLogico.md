# Ejercicio 3: Comparando dos números

Solicita al usuario dos números enteros por teclado y muestra el resultado de aplicar los seis operadores de comparación entre ellos:

- ¿Son iguales? (`==`)
- ¿Son diferentes? (`!=`)
- ¿El primero es mayor que el segundo? (`>`)
- ¿El primero es menor que el segundo? (`<`)
- ¿El primero es mayor o igual que el segundo? (`>=`)
- ¿El primero es menor o igual que el segundo? (`<=`)
