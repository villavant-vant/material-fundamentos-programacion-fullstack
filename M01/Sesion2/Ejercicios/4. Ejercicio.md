# Ejercicio 4: Registro de estudiante

Construye un programa de registro de estudiante que:

1. Solicite por teclado: nombre completo, edad, tres notas, género (M/F) y si está matriculado (true/false).
2. Calcule el promedio de las tres notas.
3. Declare una constante `NOTA_APROBATORIA = 13`.
4. Evalúe si el estudiante aprobó (promedio >= nota aprobatoria).
5. Imprima un resumen con todos los datos y el resultado, usando `printf` para mostrar el promedio con 2 decimales.
